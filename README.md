This repository contains my submission for the QA Automation Assessment.

In **Task 1**, I performed manual security testing on the Conduit demo application and identified five security vulnerabilities related to authentication, session management, and account protection. A root-cause analysis was also conducted for the No Rate Limiting issue.

In **Task 2**, I built an automated cryptocurrency monitoring workflow using n8n. The workflow retrieves data from the CoinGecko API, filters the top cryptocurrencies, enriches the data with Bitcoin market details, applies conditional logic, and sends automated notifications to Discord.

The repository includes the task report, workflow export file, screenshots, and supporting documentation.
